# Code Preparation Repository

## 📁 Project Structure

```
preparation/
├── src/
│   ├── js/
│   │   ├── algorithms/
│   │   │   └── algorithms.js          # Sorting, searching, recursion algorithms
│   │   ├── utilities/
│   │   │   └── string-and-utility-algorithms.js  # String utilities, pattern algorithms, prototypes
│   │   └── api-service.js             # API calls and UI controller classes
│   ├── php/
│   │   └── php_oop.php               # PHP OOP examples
│   └── database/
│       └── database.sql              # SQL queries and schema
├── docs/
│   └── situations.md                 # Documentation and notes
└── README.md                         # This file
```

## 📚 File Organization

### JavaScript (`src/js/`)
- **algorithms/** - Core algorithms (bubble sort, binary search, factorial, fibonacci, etc.)
- **utilities/** - String manipulation, patterns, and utility functions
- **api-service.js** - API service and UI controller implementation

### Backend (`src/php/`, `src/database/`)
- **php_oop.php** - Object-oriented PHP code
- **database.sql** - SQL queries and database schema

### Documentation (`docs/`)
- **situations.md** - Notes, scenarios, and documentation

## 🚀 Quick Start
All code files are organized by type and functionality for easy navigation and maintenance.
